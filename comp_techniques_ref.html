EXP 1 Intermediate Value Theorem + Matrix ▼
Theory
IVT: If f(x) is continuous on [a,b] and f(a) × f(b) < 0, then at least one root exists between a and b.
Transpose: Swap rows ↔ columns. matrix[i][j] → matrix[j][i]

EXP 2 Bisection Method ▼
Idea: Repeatedly halve the interval [a,b]. Midpoint c=(a+b)/2.
If f(c)×f(a) < 0 → root in left half → b=c
Else → root in right half → a=c
c = (a + b) / 2   |   n ≥ [log(b-a) - log(e)] / log2
print("Program for Bisection method")

EXP 3 Newton-Raphson Method ▼
Idea: Start with guess x₀, improve using tangent line.
Need BOTH f(x) AND f'(x) (derivative).
x₁ = x₀ - f(x₀) / f'(x₀)   |   xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ)
print("Program for newton Raphson method")
import math

def func(x):
    return math.cos(x) - x * math.exp(x)  # f(x) — change this

def fd(x):
    return -math.sin(x) - (x * math.exp(x) + math.exp(x))  # f'(x) — change this

x0 = float(input("Enter the value of x0: "))
n = int(input("Enter the number of iteration: "))

for i in range(1, n+1):
    print("Enter the number of iteration", i)
    fx = func(x0)
    fdx = fd(x0)
    x1 = x0 - (fx / fdx)
    print("x0= ", x0, "f(x0)= ", fx, "fd(x0)= ", fdx, "x1= ", x1)
    x0 = x1
    print("The root of the equation is", x1)
print("End")
Input: x0=0, n=5 → ... → The root of the equation is 0.5177...

EXP 4 Newton's Forward Interpolation ▼
Use when: x values are equally spaced.
p = (xr - x[0]) / h    h = x[1] - x[0]
yp = y₀ + p·Δy₀ + p(p-1)/2! · Δ²y₀ + p(p-1)(p-2)/3! · Δ³y₀ + ...

EXP 5 Lagrange's Interpolation ▼
Use when: x values are NOT equally spaced.
For each j: multiply (xr-x[i])/(x[j]-x[i]) for all i≠j, then × y[j] 
# Program for Lagranges Method
x = []
y = []
n = int(input("Enter the values of entries:"))
print("Enter the value of x")
for i in range(n):
    ele = float(input())
    x.append(ele)
print(x)
print("Enter the value of y")
for i in range(n):
    ele = float(input())
    y.append(ele)
print(y)
xr = float(input("Enter the value of x for which y is to be calculated:"))
fy = 0
for j in range(0, n):
    num = 1
    den = 1
    for i in range(0, n):
        if(i != j):
            num = num * (xr - x[i])
            den = den * (x[j] - x[i])
    fy = float(fy + (num/den) * y[j])
print("The value of fy at xr=", xr, "is yr=", fy)
Entries=4, x=[1,3,4,5], y=[1,9,16,25], xr=1.5 → yr= 2.25
⚡ num builds numerator, den builds denominator for each Lagrange basis term. i≠j is the key condition.
EXP 6 Simpson's 1/3 Rule (Numerical Integration) ▼
Formula: I = h/3 × [(y₀+yₙ) + 4(odd terms) + 2(even terms)]
n must be EVEN. Odd index → ×4, Even index → ×2
I = (h/3) × [first + last + 4×(y₁+y₃+...) + 2×(y₂+y₄+...)]
Program — Version 1 (sin-log-exp)
print("Program for Simpson's 1/3 Rule:")
import math

def f(x):
    return math.sin(x)-math.log(x)+math.exp(x)

lo = float(input("Enter the lower limit:"))
up = float(input("Enter the upper limit:"))
n = int(input("Enter the number of strips:"))
h = float((up - lo) / n)
print("Step size(h):", h)
sum = 0
for i in range(1, n):
    k = lo + i * h
    if i % 2 == 0:
        sum = sum + f(k) * 2
    else:
        sum = sum + f(k) * 4
sum = float(h/3 * (sum + f(lo) + f(up)))
print("Answer:", sum)
print("End")

EXP 7 Runge-Kutta 4th Order (ODE) ▼
Solves: dy/dx = f(x,y) given y(x₀) = y₀
Computes 4 slopes K1 K2 K3 K4, takes weighted average.
K1 = h·f(x₀, y₀)  |  K2 = h·f(x₀+h/2, y₀+K1/2)
K3 = h·f(x₀+h/2, y₀+K2/2)  |  K4 = h·f(x₀+h, y₀+K3)
K = (K1 + 2K2 + 2K3 + K4) / 6  |  y₁ = y₀ + K 
# Program for Runge-Kutta method
def f(x, y):
    return x + y  # change this — dy/dx = f(x,y)

x0 = float(input("enter the value of x0:"))
y0 = float(input("enter the value of y0:"))
xr = float(input("enter the value of x at which y is to be found out"))
h = float(input("enter the number of stepsize:"))

while(x0 < xr):
    k1 = h * f(x0, y0)
    k2 = h * f(x0 + h/2, y0 + k1/2)
    k3 = h * f(x0 + h/2, y0 + k2/2)
    k4 = h * f(x0 + h, y0 + k3)
    k = 1/6 * (k1 + 2*k2 + 2*k3 + k4)
    print("k=", k)
    y1 = y0 + k
    x1 = x0 + h
    print("y1=", y1)
    print("x1=", x1)
    x0 = x1
    y0 = y1
print("end")

EXP 8 Jacobi Method (Simultaneous Equations) ▼
Solves: system of 3 equations with 3 unknowns iteratively.
Rearrange each equation for its diagonal variable. Substitute, repeat.
Condition: diagonal elements must be largest (dominant diagonal).
System from your file: 20x₁+x₂-2x₃=17, 3x₁+20x₂-x₃=-18, 2x₁-3x₂+20x₃=25
print("Program for Jacobi's Method")

def f1(x, y, z):
    return 1/20 * (17 - y + 2*z)   # x1 = (17 - x2 + 2x3) / 20

def f2(x, y, z):
    return 1/20 * (-18 - 3*x + z)  # x2 = (-18 - 3x1 + x3) / 20

def f3(x, y, z):
    return 1/20 * (25 - 2*x + 3*y)  # x3 = (25 - 2x1 + 3x2) / 20

x0 = int(input("Enter the value of x0: "))
y0 = int(input("Enter the value of y0: "))
z0 = int(input("Enter the value of z0: "))
n = int(input("Enter the number of iteration: "))
x0 = 0
y0 = 0
z0 = 0

for i in range(0, n):
    x1 = f1(x0, y0, z0)
    y1 = f2(x0, y0, z0)
    z1 = f3(x0, y0, z0)
    print("iteration no:", i+1)
    print(x1, "\t\t", y1, "\t\t", z1)
    x0 = x1
    y0 = y1
    z0 = z1
print("End")
x0=y0=z0=0, n=3 → iteration 1: 0.85, -0.9, 1.25 → ... converging
⚡ If examiner gives different equations, ONLY change f1, f2, f3 return statements. Rearrange: x = (d - other terms) / diagonal coefficient
How to rearrange any system
Given: a·x + b·y + c·z = d
Rearrange for x: x = (d - b·y - c·z) / a
Do same for y and z using their respective diagonal coefficients.
